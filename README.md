# Heart Disease Prediction System

A Machine Learning web application that predicts the likelihood of heart disease based on patient health parameters.

## Features
- Predicts heart disease risk using a K-Nearest Neighbors (KNN) model
- Performs data preprocessing and feature engineering
- Includes Exploratory Data Analysis (EDA)
- Interactive web interface built with Streamlit
- Real-time predictions based on user input

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit

## Machine Learning Workflow
1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training using KNN
6. Model Evaluation
7. Deployment with Streamlit

## Results
The model was evaluated using classification metrics such as accuracy, confusion matrix, precision, recall, and F1-score.
